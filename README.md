# ⚡ Electricity Demand Forecasting

A machine learning project that predicts next-hour electricity demand using historical data and real-time weather inputs.

## 🚀 Features
- Predicts electricity demand using ML models
- Models used: Linear Regression, Random Forest, XGBoost
- Feature engineering with lag values & moving averages
- Streamlit web app for interactive predictions
- Live weather data using OpenWeatherMap API

## 🛠 Tech Stack
Python, Pandas, Scikit-learn, Streamlit, Matplotlib

## ▶️ Run Locally
pip install -r requirements.txt  
streamlit run app.py
